SpiderOak Normally I warn against using Cloud Service for anything you want private. SpiderOak one exception, with some reservation. It’s a decent enough alternative to DropBox as it is coded with “Zero Knowledge” (so say the developer) and when you install it, a set of encryption keys is created client-side. When you upload data to SpiderOak servers, they’re encrypted on your* computer and then uploaded.
